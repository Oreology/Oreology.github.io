HighJack 1

HighJack 1 was the first high-altitude balloon that we launched, as part of the SDState Cubesat Team. It was an initial proof-of-concept and had basic circuitry enclosed in a relatively large styrofoam box, filled with hand warmers and tied to a latex helium balloon.

Flight Profile

Tracked using a GPS Spot Tracker, we did not have a GPS module at the time.

Circuitry

On our first flight we only had 3 sensors - the BME280 Barometric Pressure, Temperature sensor, the MPL3115A2 Altimeter (uses barometric pressure and temperature to estimate altitude), and the LIS3DH 3-axis accelerometer. Python was used to interface with the sensors and everytime the Raspberry Pi was turned on, a new .txt file was made that would record sensor data.
